# TeaCoffee

веб-сайт для покупки товаров связанных с кофе или чаем, данные пользователей хранятся в БД

## Требования
Что нужно установить перед запуском:
*   Node.js (версия 14+)

## Установка и запуск
1.  **Скачайте архив sait3.rar и распакуйте в удобное место**
2.  Перейдите в терминале в папку server внутри проекта: "cd путь_к_проекту/server"
3.  Установите зависимости заново: "npm install"
4.  Запустите сервер: "node server.js"
5.  войдите на сайт именно через файл mainP.html

### Примечания
при входе с логином: admin и паролем: 123, вы войдете как админ который может принимать или отклонять заказы
